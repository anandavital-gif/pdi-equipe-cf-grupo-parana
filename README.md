# PDI Equipe — Grupo Paraná

Painel de gestão de desenvolvimento individual para a equipe Contábil e Fiscal do Grupo Paraná.

## Acesso online

🔗 **[Abrir painel](https://anandacvital.github.io/pdi-equipe-grupo-parana/)**

## Funcionalidades

- **Visão geral** — métricas do ciclo e radar comparativo da equipe
- **Equipe** — cards com competências e status de PDI de cada colaboradora
- **Perfil individual** — PDI detalhado, motivadores, insights e acompanhamento de ações
- **Perfis DISC** — visualização dos perfis DISC (Innermetrix DISC Plus) de cada colaboradora, comparativo da equipe e Values Index
- **Consultoria IA** — orientação personalizada de feedback usando a API Anthropic, baseada nos perfis DISC da líder e da colaboradora selecionada
